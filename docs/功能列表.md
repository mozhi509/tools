## 二、后端 API

### 1. 工具类 API（`/api/tools`）

- **JSON**
  - `POST /api/tools/json/format`：格式化 JSON
  - `POST /api/tools/json/validate`：验证 JSON
  - `POST /api/tools/json/minify`：压缩 JSON
- **Base64**
  - `POST /api/tools/base64/encode`：Base64 编码
  - `POST /api/tools/base64/decode`：Base64 解码
- **URL**
  - `POST /api/tools/url/encode`：URL 编码
  - `POST /api/tools/url/decode`：URL 解码

### 2. 视频 API（`/api/video`）

- `POST /api/video/upload`：上传视频（最大约 100MB）
- 视频裁剪、合并、滤镜等剪辑相关接口（依赖 ffmpeg）

### 3. 分享 API（`/api/share`）

- `POST /api/share/create`：创建分享，返回短链接（数据存 Redis，24 小时有效）
- `GET /api/share/:shareId`：根据分享 ID 获取分享数据

### 4. 其他

- `GET /api/health`：健康检查

---

## 三、技术特性

- **前端**：React + TypeScript + Ant Design + React Router
- **后端**：Express，支持 CORS、Helmet、请求体大小限制、API 速率限制（如 15 分钟 100 次）
- **存储**：Redis（分享数据、会话等）
- **部署**：支持 Docker、PM2（见 `Dockerfile`、`ecosystem.config.js`）

---

## 四、说明

- 导航中出现的「哈希」「加密」等入口若暂无对应路由，则功能尚未实现，以实际路由为准。
- 分享链接在生产环境依赖 `DOMAIN`、`HTTPS_ENABLED` 等配置生成正确 URL。
